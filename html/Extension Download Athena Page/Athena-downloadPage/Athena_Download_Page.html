Download the Chrome Extension

Download

Instructions:

How to install the extension:

1. Download the zip

2. Extract the zip

3. Go to chrome://extensions/

4. Enable developer mode by checking the box.

5. click "load unpacked extension"

6. load the folder "chrome_extension_popup" you extracted from the zip. NOTE: sometimes the extraction will cause that folder to have a folder of the same name inside it. in that case, load the inside folder.

7. An Athena icon should appear along with the rest of your chrome extensions.

How to use the extension:

8. Go to Accessibility Or Atos

For example, This is the how the Accessibility page option bar should look now:

9. Now click on the Athena extension icon. This popup window should open:

10. in the small popup that appears, Click on 'change tooltip'.

This is how the same Accessibility page option bar should look:

11. Some elements in the page now have an attached (hidden) tooltip. Place your mouse pointer on the 'Home' Button.

Alternatively, you can 'Tab' to it or press SHIFT+H on your keyboard.
