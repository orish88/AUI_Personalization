Download the Chrome Extension

Download

Instructions:

How to install the extension:

1. Download the zip

2. Extract the zip

3. Go to chrome://extensions/

4. Enable developer mode by checking the box.

5. click "load unpacked extension"

6. load the folder "chrome_extension_popup" you extracted from the zip

7. An Athena icon should appear along with the rest of your chrome extensions.

How to use the extension:

8. Go to Accessibility

9. press the Athena extension icon

10. in the small popup, enter a profile url, like:https://rawgit.com/orish88/AUI_Personalization/master/profiles/test_profile1.json

11. Click on the "Click to personalize page" Button.
